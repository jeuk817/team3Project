3.1절 100주년 기념 카드뉴스

Project information

  • Category: Card News
  • Client: 대검찰청
  • Project date: 14 July, 2019
  • Project URL: 대검찰청 블로그

대검찰청 블로그 기자단 첫 카드뉴스프로젝트였습니다. 대검찰청은 딱딱한 분위기에서 탈피하고 법을 더욱 알기쉽게 국민들에게 알리는 것을 주 목표로 하는 기자단이었습니다. 그래서 새로 개정된 법을 일반 국민들이 이해하기 쉽도록 요약 정리하고 어려운 법 용어들을 쉽게 풀어 카드뉴스를 제작하였습니다.
